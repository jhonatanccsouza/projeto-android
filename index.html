Criação

Quem desenvolveu o site foi o desenvolvedor iniciante Jhonatan Correia Cunha de Souza, o site surgiu de um simples projeto (Projeto Android) do curso de HTML5 e CSS3 do CursoemVideo onde ele teve o primeiro contato com o desenvolvimento de sites.

Projeto Android

O Projeto Android foi o primeiro projeto envolvendo HTML5 e CSS3 que ele desenvolveu. Criado por seu professor Gustavo Guanabara o Projeto Android foi um grande desafio para ele começar a ter uma noção de como seria desenvolver sites.

Após a criação da página do Projeto Android, ele decidiu que iria aprimorar o site criando uma página inicial que é esta que você esta neste momento
